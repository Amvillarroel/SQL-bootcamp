# ADR-001 — Alcance del Proyecto

**Estado:** Aprobado
**Fecha:** Julio de 2026

---

# Contexto

El proyecto nace con el objetivo de construir una base de datos sobre la cual desarrollar un Bootcamp completo de SQL.

Era necesario definir si el alcance estaría orientado a un ejemplo académico simplificado o a un modelo inspirado en un entorno empresarial.

---

# Decisión

El proyecto representará una empresa mediana de comercialización de productos tecnológicos, utilizando procesos similares a los de un ERP real, pero evitando la complejidad innecesaria de una solución comercial completa.

---

# Consecuencias

* El modelo será suficientemente amplio para desarrollar desafíos SQL de todos los niveles.
* El dominio del negocio permanecerá estable durante todo el Bootcamp.
* La documentación podrá evolucionar sin cambiar el contexto empresarial.
