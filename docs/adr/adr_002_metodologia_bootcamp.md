# ADR-002 — Metodología del Bootcamp

**Estado:** Aprobado
**Fecha:** Julio de 2026

---

# Contexto

Era necesario definir una metodología que permitiera desarrollar el proyecto de forma ordenada, evitando mezclar etapas de análisis, diseño e implementación.

---

# Decisión

El Bootcamp se desarrollará por módulos secuenciales.

Cada módulo deberá completarse, revisarse y aprobarse antes de iniciar el siguiente.

La documentación será elaborada con un enfoque profesional y formará parte del entregable final del proyecto.

---

# Consecuencias

* Se reducirá el riesgo de rehacer trabajo por decisiones prematuras.
* Cada etapa contará con una línea base claramente definida.
* El repositorio reflejará la evolución real del proyecto.
